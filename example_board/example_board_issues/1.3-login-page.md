# 1.3 — Build Login Page

**Status:** in progress
**Files:** `pages/login.php`, `templates/login.tpl`, `js/auth.js`

## What to build

Login form with email/password fields, client-side validation, and error display.

## Form fields

- **Email**: required, validate format client-side
- **Password**: required, min 8 chars
- **Remember me**: checkbox, extends session to 30 days

## Error handling

- Invalid credentials → red alert: "Invalid email or password"
- Rate limited → "Too many attempts. Try again in 1 minute."
- Network error → "Could not reach the server"

## Notes

- Submit via `fetch()` to `/api/auth/login`, no full page reload
- On success redirect to `/dashboard`
- Store token in `httpOnly` cookie (not localStorage)
