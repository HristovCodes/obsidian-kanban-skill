# 1.2 — Add API Endpoints

**Status:** done
**Files:** `api/users.php`, `api/auth.php`, `.htaccess`

## Endpoints

| Method | Route | Purpose |
|--------|-------|---------|
| POST | `/api/auth/login` | Authenticate, return session token |
| POST | `/api/auth/logout` | Invalidate session |
| GET | `/api/users` | List users (admin only) |
| GET | `/api/users/{id}` | Get user details |
| PUT | `/api/users/{id}` | Update user profile |

## Auth flow

1. POST `/api/auth/login` with `{email, password}`
2. Verify bcrypt hash against `users.password_hash`
3. Create `sessions` row, return `{token, expires_at}`
4. Subsequent requests send `Authorization: Bearer {token}`

## Notes

- All endpoints return `{"success": true/false, "data": {...}, "error": "..."}`
- Rate limit login to 5 attempts per minute per IP
